# CORE PRINCIPLE

- progress
- reward
- momentum

- gambling

- unrealistic payouts

# SYSTEM DESIGN

## 1. VIRTUAL ECONOMY (NOT REAL MONEY)

Create:
- "Clarity Points" (XP)
- "Wealth Score" (progress metric)

Users earn points for:
- logging in daily
- completing actions
- engaging with tools

---

## 2. DAILY SPIN SYSTEM (SAFE VERSION)

Replace cash rewards with:

- XP boosts
- feature unlocks
- insights
- streak bonuses

Example rewards:

- +500 XP
- +2x XP for next session
- unlock advanced insight
- streak protection

---

## 3. PROGRESSION SYSTEM

Users start with:
- a base financial simulation profile

Progress through:
- levels
- milestones
- achievements

---

## 4. LEADERBOARD (OPTIONAL)

Show:
- top progress users
- weekly rankings

DO NOT:
- tie to real money

---

## 5. STREAK SYSTEM

Reward:
- daily logins
- consistency

Example:
- 3-day streak → bonus XP
- 7-day streak → unlock feature

---

## 6. FIRST USER ADVANTAGE (SAFE VERSION)

Instead of giving money:

First 10,000 users get:
- "Founder Status"
- permanent XP multiplier
- exclusive badge

---

## 7. YEARLY REWARD (SAFE STRUCTURE)

Instead of random payout:

Offer:
- structured contest
- based on engagement or milestones

Example:
- top contributors win prize

MUST:
- include clear rules
- comply with sweepstakes laws

---

## 8. PSYCHOLOGY LAYER

Use:

- progress bars
- level-ups
- small wins
- visible improvement

Avoid:
- fake jackpots
- misleading odds
